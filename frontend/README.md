# Frontend

This is the frontend application built with React, TypeScript, and modern Web3 libraries.

## How to run

- Install: `npm install`
- Run dev mode: `npm run dev`

## Libraries used

- **Tailwind CSS**: [Documentation](https://tailwindcss.com/docs)
- **React Query (@tanstack/react-query)**: [Documentation](https://tanstack.com/query/latest)
- **Wagmi**: [Documentation](https://wagmi.sh/react/getting-started)
- **Viem**: [Documentation](https://viem.sh/docs/getting-started)
- **RainbowKit**: [Documentation](https://www.rainbowkit.com/docs/introduction)
